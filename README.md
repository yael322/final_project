# final_project

## Инструменты для реализации:
> python,
> telegram bot,
> sqlite для хранения данных

## Проект состоит из:
> разработки Telegram-бота по игре HSR,
> создания удобного меню с интерактивными кнопками,
> реализации просмотра информации о персонажах,
> красивого оформления ответов с эмодзи и игровым стилем,
> тестирования и отладки,
> подготовки документации по использованию.

## Референсы для проектя
> просмотр персонажей

## План работы
> Мой план работы состоит из того что я буду брать части от других проектов которых я уже закончила и изменять их под свой финальный проект чтобы уменьшить время работы и силы. в самом проекте будет начинать все команда "start" которая будет расприделять на разделы: Раздел «Информация о игре», Раздел «Персонажи». В проекте будет База данных персонажей которая будет иметь информацию о персонаже который пользователь захочет посмотреть в разделе «Персонажи». Сам проект будет красиво оформлен с эмодзи и игровым стилем под саму игру.
